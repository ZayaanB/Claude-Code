# Memory System

This directory is a **symlink** to `~/claude_ai/config/memory/`.

All persistent memories about you, your preferences, and your projects are stored there.
Memories are stored as individual markdown files and indexed in MEMORY.md.

## Types of memories:
- **user**: About your role, goals, and preferences
- **feedback**: Guidance on how I should approach work
- **project**: Information about ongoing work and deadlines
- **reference**: Pointers to external resources
