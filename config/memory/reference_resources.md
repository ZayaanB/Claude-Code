# Reference Resources

## Computer Systems (C/Linux)
- [CS:APP](https://csapp.cs.cmu.edu/) — *Computer Systems: A Programmer's Perspective* (Bryant & O'Hallaron) — the standard for systems programming, heavy on C and x86
- [OSTEP](https://ostep.org/) — *Operating Systems: Three Easy Pieces* — free online textbook, very readable
- [Modern C](https://moden-c.gustedt.net/) — *Modern C* (Jens Gustedt) — free PDF, rigorous and up-to-date C standard
- [UToronto Systems Notes](https://www.cs.utoronto.ca/~strider/LectureNotes.html) — Systems programming lecture notes

## C++
- [cppreference.com](https://en.cppreference.com/) — authoritative, precise documentation
- **"A Tour of C++"** (Bjarne Stroustrup) — concise, written by the creator
- **"C++ Concurrency in Action"** (Anthony Williams) — for systems/performance work

## Python (Academic Depth)
- **"Fluent Python"** (Luciano Ramalho) — deep dive into Python's data model and internals
- [Python Data Model](https://docs.python.org/3/reference/datamodel.html) — official docs on Python's object system

## Algorithms & Theory
- **"Introduction to Algorithms"** (CLRS) — standard graduate algorithms text
- **"Algorithm Design"** (Kleinberg & Tardos) — intuitive with strong proof techniques

## TypeScript
- [TypeScript Handbook](https://www.typescriptlang.org/docs/handbook/intro.html) — official, comprehensive
- **"Programming TypeScript"** (Boris Cherny) — balances rigor with practical patterns

## Workflow
- Git/GitHub workflow documentation for collaboration
